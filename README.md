# IP Addressing and Subnetting Implementation

## Project Overview

This project demonstrates the implementation of IP addressing and subnetting to efficiently manage network resources.

The project includes a Python-based subnet calculator and a Cisco Packet Tracer simulation.

## Features

- IPv4 Address Planning
- Subnetting Implementation
- Network Segmentation
- Python Subnet Calculator
- Cisco Packet Tracer Simulation

## Technologies Used

Python
Cisco Packet Tracer
Computer Networking Concepts

## Network Design

Base Network: 192.168.10.0/24

Departments:
HR
Finance
IT
Guest Network

Each department is assigned a subnet for better network management and security.

## How to Run Python Tool

Run:

python subnet_calculator.py

Enter the network and subnet prefix to calculate subnets.

## Simulation

The network topology was implemented using Cisco Packet Tracer.
